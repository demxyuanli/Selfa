# 截图目录

将应用截图放在此目录中，用于主 README.md 文件。

## 所需截图

1. **main-interface.png**: 显示工作区的主应用界面
2. **kline-chart.png**: 带有技术指标的K线图
3. **technical-analysis.png**: 技术分析面板
4. **prediction-analysis.png**: 预测分析功能
5. **favorites-heatmap.png**: 收藏夹热力图视图
6. **compare-analysis.png**: 对比分析功能
7. **AI-analysis.png**: AI分析预测功能

## 截图指南

- 推荐分辨率：1920x1080 或更高
- 格式：支持透明度的 PNG
- 文件大小：每张图片保持在 2MB 以下
- 命名：使用小写字母和连字符（kebab-case）
